## v5.1 修复章节计数和状态显示问题
- **问题**: 用户反馈采集速度和章节计数不对
- **根因**: popup.js `chapterDone` 处理中有 `+1` 导致章节号始终偏移1；翻页等待阶段状态文本与进度不匹配

  3. 帧计数在翻页时显示 "翻页中..."
  4. 新页面加载时显示等待状态
  5. 章节完成后帧计数显示完成标记

## v5.2 增加URL日志定位跳章问题
- **问题**: 用户反馈"应该采集第二章的时候直接跳到了第三章"
- **根因**: `pageTitle` 在循环外只获取一次，导致所有章节复用同一标题；缺乏翻页验证
- **修复**:
  1. pageTitle 改为每次循环内重新获取
  2. 增加翻页前后 URL 日志记录
  3. URL 未变化时输出警告
- **提交**: `5464eda`
