### 技术要点记录
- Service Worker 中 URL.createObjectURL 不可用，必须直接传 data URL 给 downloads API
- Chrome API 名称拼写容易出错，建议用 heredoc 或脚本生成代码
- Fine-grained PAT 无法创建 GitHub 仓库，需用 gh CLI 或 Classic PAT
- 小说站常用自定义 overflow 容器，window.scrollBy() 可能无效
